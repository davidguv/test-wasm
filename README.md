# test-wasm
Python script for execute wasm in chrome and export the console log to a output file

## System Requirements

* Python3
* Google Chrome
* [ChromeDriver](https://chromedriver.chromium.org/)
* [Selenium for python](https://pypi.org/project/selenium/)
